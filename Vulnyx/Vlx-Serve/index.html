Vlx_Serve

靶机来源： https://vulnyx.com/

难度：Easy

思维导图：

一、信息收集

1、主机探测

靶机IP：192.168.2.10

2、端口扫描

1.全端口扫描

开放端口：22、80

2.详细信息扫描

3.udp扫描

二、WEB渗透

1、80端口

访问80端口，是Apache的默认页面

2、目录扫描

找到目录：/notes.txt、/secrets、/webdav

查看/notes.txt

提示我们用户teo的数据库文件在secret目录下面，同时需要将X修改成数字

对/secrets目录进行递归扫描，由于是数据库文件，因此扫描的时候只需要扫描数据库文件后缀的即可，列如：.sql,.db,.sqlitb,.kbdx

找到数据库文件db.kdbx

3、kdbx文件

.kdbx 文件是 KeePass 密码管理器专用的加密数据库文件，因此该文件是经过加密的，需要进行破解，使用john进行爆破

找到db.kdbx的密码是dreams

直接使用在线工具打开 成功获取到账号和和密码：admin:w3bd4vXXX

根据提示信息，需要将XXX修改为数字

4、HTTP-Bruf

访问webdav是需要进行登录的，根据获取到的信息使用hydra进行爆破

成功获取到密码：w3bd4v513

5、Webdav

使用凭证成功进行登录 但是登录后无任何的内容

WebDAV是基于 HTTP 协议的高级文件管理器，标准的 HTTP 主要用于“看”网页（下载），而 WebDAV 则允许你像操作本地硬盘一样，对远程服务器上的文件进行读、写、移动、复制和锁定

因此可以上传文件到服务器，使用curl进行上传

成功将文件上传到服务器

三、获取shell

kali开启监听后，访问rev.php，成功获取到shell

稳定shell

四、权限提升

1、获取teo权限

查看sudo -l

使用GTFOBins的方法，无法进行利用

使用upload方法，将id_rsa上传到本地

成功获取到id_rsa

使用RASCrack进行破解

成功获取到id_rsa的密码:private

成功获取到权限

2、获取root权限

查看sudo -l

查看使用方法

功能与man类似
