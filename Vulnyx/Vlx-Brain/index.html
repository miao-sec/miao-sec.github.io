三、获取shell

方法一、/proc/sched_debug

/proc/sched_debug 是 Linux 内核提供的一个虚拟调试文件，位于 /proc 伪文件系统中。它用于输出内核调度器（主要是 CFS，即 Completely Fair Scheduler）的详细运行状态和调试信息，是系统性能分析、调度问题排查和内核调试的重要工具。 可以列出所有的进程任务

在进程任务中，能成功获取到ben用户的登录凭证

直接进行登录即可

方法二、PHP伪协议

尝试使用PHP伪协议去读取index.php源码

进行解密

使用FilterChain进行获取shell 工具：GitHub - synacktiv/php_filter_chain_generator · GitHub

  1. 构造代码
  1. 执行反弹shell的命令
  2. 监听，成功获取到shell
