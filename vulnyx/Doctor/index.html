Doctor

靶机说明

https://vulnyx.com/file/Doctor.php

一、主机探测

IP地址：192.168.2.44

二、端口扫描

1、全端口扫描

开放端口：22和80

2、详细信息扫描

3、UDP端口扫描

没有端口开放

三、WEB渗透

访问80端口，发现该系统模板是Colorlib，同时找到一个域名contact.com

追加域名到hosts文件中

1、目录扫描

使用IP和域名进行目录扫描，都没有找到有用的目录

2、文件包含

在访问时，发现doctor页面存在文件包含

尝试读取/etc/passwd，发现可以读取，同时找到一个用户admin

读取admin的ssh私钥

  • 先读取authorized_keys
  • 再读取私钥文件id_rsa

四、SSH私钥密码破解

发现私钥是经过加密的，需要进行破解

找到私钥的密码unicorn

五、获取USER权限

直接使用私钥进行连接

USER FLAG

六、权限提升

使用sudo -l，提示无法使用

passwd写权限

查看文件权限，发现/etc/passwd具有w权限

写入一个具有root权限的用户

  • 使用openssl生成密码123456的哈希值

  • 写入/etc/passwd

切换用户，发现具有root权限

ROOT FLAG
