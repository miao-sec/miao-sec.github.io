【总结】

1、Nginx配置错误

如果静态文件存储在 /home/ 目录下，而该目录在url中名字为files，那么就需要用alias设置目录的别名。Nginx在配置别名(Alias)的时候，如果忘记加 / ，将造成一个目录穿越漏洞。

  • 产生原因：由于 location 中目标路径未闭合导致相对路径穿越

目录穿越后，是 Alias 目标目录的上一层，并且只能穿越一层目录。

修复：闭合 location 路径即可
